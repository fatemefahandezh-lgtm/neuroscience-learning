## Key structures reviewed

### Spinal cord
- Main pathway for sensory input and motor output.
- Responsible for basic reflexes.

### Medulla oblongata
- Controls vital autonomic functions.
- Breathing, heart rate, and basic survival mechanisms.

### Cerebellum
- Coordination of movement.
- Balance, timing, and motor learning.

### Brainstem (overview)
- Connects spinal cord to higher brain areas.
- Essential for life-sustaining functions.

### Cerebral cortex (introductory view)
- Higher cognitive functions.
- Sensory processing, motor planning, and integration.
